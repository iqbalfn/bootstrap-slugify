Slugify

Convert one input form control value to other input form control as slug version.

Overview

Things to know when using the slugify helper

  1. The bootstrap version must be greater then or equal to 4.2
  2. The value taken from other input, and converted while typing.
  3. Auto update of main input will only work if the input has empty value and never been focused.
